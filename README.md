# VPS Manager

Персональный инструмент для управления и мониторинга VPS серверов.

## Проблема

Когда у тебя 20+ серверов на разных хостингах с разными датами оплаты — это хаос. Платные альтернативы существуют, но зачем платить, если можно сделать своё?

## Что делает приложение

- **Организация**: папки для группировки серверов (по проектам, хостингам, etc.)
- **Учёт**: IP, название, дата оплаты, цена
- **Мониторинг**: автоматический пинг для проверки доступности
- **[Будущее]**: агент на серверах для сбора метрик (CPU, RAM, диск)

## Стек

| Компонент | Технология |
|-----------|------------|
| Frontend | React + Vite + TypeScript |
| Backend | Python + FastAPI |
| Database | PostgreSQL |
| ORM | SQLAlchemy 2.0 + Alembic |
| Containers | Docker + Docker Compose |

## Подход к разработке

> Это pet-проект для обучения, а не гонка за результатом.

**Принципы:**
1. **Понимание важнее скорости** — разбираемся в каждой части, не копируем бездумно
2. **Итеративность** — сначала минимум, потом улучшаем
3. **Без over-engineering** — никакой авторизации, пока не нужна
4. **Docker-first** — всё в контейнерах с первого дня

**Текущий этап:** базовая структура + docker-compose

## Запуск (dev)

```bash
# Поднять всё
docker-compose up -d

# Только база (для локальной разработки)
docker-compose up -d db

# Логи
docker-compose logs -f backend
```
